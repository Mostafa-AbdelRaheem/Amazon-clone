# Build Tool

- React

# Installation

### Step 1: npm and node are installed from the terminal.

```
npm -v

node -v
```

### Step 2 : npm install

in order to get all the dependencies we need to install npm

```
npm install
```

# Run the project

```
npm run dev
```
